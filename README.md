 Book Management API

This is a simple RESTful API built with Node.js and Express.js to manage a collection of books. You can create, read, update, and delete (CRUD) book entries.

  Features
- GET all books
- POST a new book
- PUT (update) a book by ID
- DELETE a book by ID

 Technologies Used
- Node.js
- Express.js
- Postman (for testing)
